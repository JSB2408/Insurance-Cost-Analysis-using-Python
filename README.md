## Overview
This project analyzes medical insurance charges using regression-based machine learning models. It studies how factors such as age, BMI, smoking status, gender, number of children, and region affect insurance costs.

The project emphasizes correct data preprocessing, feature engineering, and model evaluation.

---

## Workflow
- Data cleaning and handling missing values  
- Encoding categorical variables (gender, region, smoker)  
- Feature scaling using StandardScaler  
- Polynomial feature generation  
- Model training using Linear Regression and Ridge Regression  
- Model evaluation using R² score
